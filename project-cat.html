UX Projects

CAFÉ RÅG

About the project

Café RÅG (swedish for rye) is a newly opened local café here in Uddevalla and the owner was in need for a website.

Background

This project was part of my Web Content Manager & Designer education. Main focus during the whole project was user experience. It did however involve many different steps since the goal was to deliver a complete and fully functioning website, optimized for the unique users a.k.a the customers of the café.

About the project

University of Borås set me out on a mission to create a website for a fictional company. I named it Studio52 because it's a design bureau (still fictional) solving problems every week of the year.

Background

For this project I was supposed to design and create a website with responsive design. When designing, special care should be given to form, usability, availability, purposes and target audience. All of the website had to be created by me myself, including all code, copy text and what else.
